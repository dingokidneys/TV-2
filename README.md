This is an android app that helps UK travellers locate an appropriate TV signal.

The phone's location is used to identify a list of the strongest TV transmitter signals available.  
The relative strength of each transmitter is given with the strongest signal being shown as 100%.

Choosing a transmitter from the list displays a compass pointer in the direction of the chosen transmitter.  
The vertical or horizontal polarisation of the signal is shown in the centre of the compass.

Comments for defects or enhancements are welcome.
